## 三、注册页设计

### 3.1 字段设计

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 用户名 | 文本 | ✅ | 4-20字符 |
| 密码 | 密码 | ✅ | 至少6位 |
| 确认密码 | 密码 | ✅ | 二次确认 |
| 行业 | 下拉选择 | ✅ | 必选 |
| 公司名称 | 文本 | ❌ | 选填 |
| 联系人 | 文本 | ❌ | 选填 |
| 手机号 | 手机 | ❌ | 选填 |

### 3.2 行业选项（Demo）
- 快递物流
- 专线运输
- 仓储服务
- 跨境物流
- 冷链物流
- 物流设备

### 3.3 交互设计
- 行业选择后立即关联推荐策略
- 注册成功自动登录并跳转首页

## 五、AI对话推荐机制

### 5.1 对话入口
- 首页右下角固定悬浮球："有问题？问我"
- 点击展开AI对话窗口

### 5.2 画像更新逻辑

```
用户输入 ──▶ NLP解析 ──▶ 提取标签 ──▶ 更新画像 ──▶ 刷新推荐
                           │
                           ▼
                    标签类型：
                    • 业务需求（整车/零担/仓储）
                    • 痛点（时效/成本/管理）
                    • 意向（咨询/试用/购买）
```

### 5.3 示例对话

| 用户说 | 提取标签 | 推荐调整 |
|-------|---------|---------|
| "我们经常发整车" | [整车运输] | 增加整车管理系统推荐 |
| "时效总是把握不好" | [时效痛点] | 推荐路线优化、智能调度 |
| "想了解下你们WMS" | [意向:仓储] | 置顶WMS产品，增加案例 |

### 5.4 推荐刷新时机
- 每次对话结束后自动刷新"猜你喜欢"区域
- 显示提示："根据您的需求，推荐已更新"
